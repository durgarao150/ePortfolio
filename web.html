Web Development Projects

Solar System Quiz App

This Solar System Quiz App is an engaging educational tool designed to test and expand users' knowledge about planets, moons, and other celestial objects. The app combines interactive features with visually appealing design to make learning about the solar system fun and accessible.

The quiz interface was redesigned to improve usability and engagement by replacing dropdowns with interactive buttons. The new design includes polished borders and box-shadow effects for a modern look. Buttons now feature hover effects and clear visual feedback: selected options highlight in light blue, correct answers turn green, and incorrect ones turn red after verification.

Drag-and-Drop: Country to Continent Mapping

This interactive drag-and-drop web page allows users to map countries to their respective continents, combining an engaging activity with an educational goal. Key changes include a redesigned header, expanded content, and improved feedback messages for completed tasks.

CSS updates included rounded borders for a polished look, new colors for the background and options, and shadow effects to all boxes. The page was also center-aligned for a cleaner and more cohesive layout.
