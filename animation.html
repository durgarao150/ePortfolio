Lunar Cycle Animation

The lunar cycle refers to the periodic phases of the Moon as it orbits Earth, including phases like New Moon, First Quarter, Full Moon, and Last Quarter. This animation illustrates the complete lunar cycle in a visually engaging and educational format.

Project Overview

This animation project was created using Adobe Animate to visually explain the lunar cycle. I started with a clean vector image of the Moon and drew multiple frames to depict the phases, adding intermediate stages for smooth transitions. Motion tweening was applied to replicate the Moon's orbital movement dynamically. Narration was recorded and synced using iMovie to enhance the animation's educational value.
